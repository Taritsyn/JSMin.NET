JSMin for .Net
==============

A .NET port of the [Douglas Crockford's JSMin](http://github.com/douglascrockford/JSMin).

#Installation
This library can be installed through NuGet - [http://nuget.org/packages/JSMin.NET](http://nuget.org/packages/JSMin.NET).
